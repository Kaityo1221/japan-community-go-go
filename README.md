# 日本コミュニティイベントガイド β

DFW POGO のような大型イベント参加者向けポータルを、日本向けに作り直すための静的WebアプリMVPです。

## 入っている機能

- トップページ
- Start Here
- 当日の流れ
- 今日の攻略カード
- 前日/当日チェックリスト
- 地域別グループ一覧
- ミートアップビンゴ
- 運営向けメモ
- 共有文コピー
- ダークモード
- localStorage によるチェック状態保存

## 使い方

1. `index.html` をブラウザで開く
2. 文言や地域データは `data.js` を編集
3. 見た目は `styles.css` を編集
4. 公開する場合は、このフォルダを Netlify / Vercel / GitHub Pages などにアップロード

## 編集ポイント

### イベント情報

`data.js` の `event` を編集してください。

### 地域別グループ

`data.js` の `groups` を編集してください。
`campfireUrl` と `discordUrl` は本番URLに差し替えてください。

### ビンゴ内容

`data.js` の `bingoItems` を編集してください。
25マス中、中央は自動的に FREE になります。
