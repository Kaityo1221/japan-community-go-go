# GO FEST JAPAN 2026 — 非公式コミュニティガイド フルデモ

海外の大型コミュニティガイド型サイトを参考に、日本向けの提案デモとしてゼロから作成した静的Webアプリです。

## 使い方

1. ZIPを解凍します。
2. `index.html` をブラウザで開きます。
3. 内容を変える場合は `data.js` を編集します。

## 収録機能

- トップページ
- 土曜 / 日曜の切り替え
- Start Here（まず見る）
- レイド攻略カード
- 時間帯別スケジュール
- 一日を通して意識すること
- 日本向けフィールドTips
- 地域別CAグループ一覧
- 前日 / 当日チェックリスト
- 周辺情報（休憩・飲食・宿泊）
- CP壁紙ジェネレーター
- ミートアップビンゴ
- フォトハント
- 運営向けメモ
- リョータさん向け説明文コピー
- ダーク / ライト切替
- 文字サイズ切替
